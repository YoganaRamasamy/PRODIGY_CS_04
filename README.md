# Simple Keylogger (Educational)

## Description
This project demonstrates how keyboard inputs can be captured using Python.

## Features
- Logs keystrokes
- Saves to a file
- Stops with ESC key

## Technologies Used
- Python
- pynput library

## Ethical Considerations
This program is for educational use only. It should never be used without user consent.

## How to Run
1. Install pynput:
   pip install pynput

2. Run:
   python main.py
